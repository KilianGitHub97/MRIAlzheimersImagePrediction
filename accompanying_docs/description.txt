Context
The Data is hand collected from various websites with each and every labels verified.

Content
The data consists of MRI images. The data has four classes of images both in training as 
well as a testing set:

1.Mild Demented
2.Moderate Demented
3.Non Demented
4.Very Mild Demented

Inspiration
The main inspiration behind sharing this Dataset is to make a very highly accurate 
model predict the stage of Alzheimers.
